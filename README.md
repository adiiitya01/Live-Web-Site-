# ♻️ ReUse Market

ReUse Market is a full-stack marketplace web application designed for buying and selling second-hand items locally.  
The platform allows users to list products, explore listings, communicate with buyers/sellers, and manage their activities through a user dashboard.

---

## 🚀 Overview

This project is built to provide a simple and efficient platform for users to reuse and resell products.  
It combines frontend design with backend functionality such as authentication, product management, and user interaction.

---

## ✨ Features

- 🔐 User Registration & Login System  
- 🛍️ Browse Products by Category  
- 📦 Post Ads with Product Details  
- 👤 User Dashboard for Managing Listings  
- 💬 Chat System for Buyer-Seller Communication  
- ❤️ Wishlist / Saved Items  
- 🛡️ Admin Panel for Managing Users & Listings  
- 🚩 Report System for Products  
- 📱 Responsive Design

## ⚙️ How to Run

### Frontend Only
1. Download or clone the repository  
2. Open the project folder  
3. Run `index.html` in your browser  

### Full Project (Backend)

1. Clone the repository  
git clone https://github.com/adiiitya01/Live-Web-Site-.git  

2. Open the folder  
cd Live-Web-Site-  

3. Install dependencies  
pip install flask flask-login flask-sqlalchemy flask-socketio  

4. Run the application  
python app.py
